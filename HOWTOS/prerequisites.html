Supported platforms

This release candidate supports iPAQ H3600, H3700, H3800, H3900, H5100, H5400, and H5500 units.

H51xx, H54xx, H55xx: The bootldr does not support the display or CF/PCMCIA, so the root images must be installed via serial port.

There is experimental support for iPAQ H2200, H6300 and hx4700 units in v0.8.4.

Note that, unfortunately, the Familiar installation manual does not yet cover these devices. If you have followed the corresponding ports projects and are familiar with the installation procedures, you are encouraged to try this release candidate. Otherwise you are recommended to wait for the final v0.8.4 release which will be accompanied by the required documentation.

The iPAQ H1900 is not yet supported. Please refer to the H1900 project page for current status.

Hardware requirements

As well as the iPAQ itself, you will need either:

  • a CompactFlash memory card and a suitable sleeve; or
  • a host machine running a terminal emulator, connected to the iPAQ with a serial cradle or cable.
    Note: a USB-only connection is not sufficient.

Some form of Internet connectivity is strongly recommended. Familiar supports these types of network connection:

  • CF/PCMCIA Ethernet
  • CF/PCMCIA wireless LAN
  • PPP over a serial line
  • USB connection to a host PC (requires special drivers on the host)
  • LAP or PAN connection using Bluetooth
